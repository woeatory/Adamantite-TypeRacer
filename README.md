# Adamantite-TypeRacer

CLI-based typing game

Features:

- single/multiplayer
- creating public/private rooms to compete with other players
- saving results in database

# How to start app

#### Install dependencies

```bash
go mod tidy
```

### Boot server with
```bash
go run .\server\
```

![img.png](img.png)


### Start CLI client

```bash
go run .\client\
```
